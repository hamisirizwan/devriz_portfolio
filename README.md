## Run Locally

Clone the project

```bash
  git clone https://github.com/hamisirizwan/devriz_portfolio
```

Go to the project directory

```bash
  cd devriz_portfolio
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm start
```
